ぐるなびAPIを用いたレストラン検索アプリ

開発環境: Google Chrome 65,  xampp

開発言語： php7.1


<機能一覧>
・レストラン検索　: ぐるなびAPIを使用して、現在地周辺の飲食店を検索する。
　　　　　　　　　　検索範囲、検索場所を指定可能

・レストラン情報取得 : ぐるなびAPIを使用して、飲食店の詳細情報を取得する。


<画面一覧>
・検索条件入力画面  : 条件を指定してレストランを検索する。

・検索結果画面      : 検索結果の飲食店を一覧表示する。

・店舗詳細画面      : 飲食店の詳細を表示する。


<使用しているAPI>

・ぐるなびAPI
・GeolocationAPI
